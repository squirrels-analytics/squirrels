Configuration class for setting up OAuth authentication providers.

This class can be imported from the `squirrels.auth` or the `squirrels` module.

## Constructor 

Creates a `ProviderConfigs` object.

```python
def __init__(
    self, *, client_id: str, client_secret: str, server_url: str, 
    server_metadata_path: str = "/.well-known/openid-configuration",
    client_kwargs: dict = {}, 
    get_user: Callable[[dict], RegisteredUser]
) -> None:
```

<Expandable title="arguments" defaultOpen>
  <ResponseField name="client_id" type="str" required>
    OAuth client ID
  </ResponseField>

  <ResponseField name="client_secret" type="str" required>
    OAuth client secret
  </ResponseField>

  <ResponseField name="server_url" type="str" required>
    URL of the OAuth server
  </ResponseField>

  <ResponseField name="server_metadata_path" type="str" default="/.well-known/openid-configuration">
    Path to the OAuth server metadata
  </ResponseField>

  <ResponseField name="client_kwargs" type="dict" default="{}">
    Additional keyword arguments for the OAuth client
  </ResponseField>

  <ResponseField name="get_user" type="Callable[[dict], RegisteredUser]" required>
    Function to convert OAuth user info to [RegisteredUser](/references/python/auth/RegisteredUser)
  </ResponseField>
</Expandable>

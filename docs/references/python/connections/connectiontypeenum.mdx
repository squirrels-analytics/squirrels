Enumeration of supported connection types in Squirrels.

## Values

### SQLALCHEMY

SQLAlchemy-based connections using standard database drivers.

**Value:** `"sqlalchemy"`

### CONNECTORX

ConnectorX-based connections for fast data loading.

**Value:** `"connectorx"`

### ADBC

Arrow Database Connectivity (ADBC) connections.

**Value:** `"adbc"`

### DUCKDB

Native DuckDB connections.

**Value:** `"duckdb"`

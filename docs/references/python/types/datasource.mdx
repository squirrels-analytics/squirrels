---
title: "DataSource"
tag: "class"
description: "Abstract base class for data sources"
---

Abstract base class for all data source classes. Data sources populate parameter options from lookup tables.

This is an abstract class and should not be instantiated directly. Use one of the concrete implementations:
- `SelectDataSource`
- `DateDataSource`
- `DateRangeDataSource`
- `NumberDataSource`
- `NumberRangeDataSource`
- `TextDataSource`

See the `data_sources` module for concrete implementations.

---

Abstract base class for parameter configuration classes. Used internally by the Squirrels framework.

## Attributes

<ResponseField name="name" type="str">
  The name of the parameter
</ResponseField>

<ResponseField name="label" type="str">
  The display label for the parameter
</ResponseField>

<ResponseField name="description" type="str" default="">
  Explains the meaning of the parameter
</ResponseField>

<ResponseField name="user_attribute" type="str | None" default="None">
  User attribute that may cascade the options
</ResponseField>

<ResponseField name="parent_name" type="str | None" default="None">
  Name of parent parameter for cascading
</ResponseField>

## Methods

### copy

Creates a shallow copy of the configuration. Used for unit testing only.

**Returns:** Copy of the configuration

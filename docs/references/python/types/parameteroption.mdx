---
title: "ParameterOption"
tag: "class"
description: "Abstract base class for parameter options"
---

Abstract base class for all parameter option classes.

This is an abstract class and should not be instantiated directly. Use one of the concrete implementations:
- `SelectParameterOption`
- `DateParameterOption`
- `DateRangeParameterOption`
- `NumberParameterOption`
- `NumberRangeParameterOption`
- `TextParameterOption`

See the `parameter_options` module for concrete implementations.

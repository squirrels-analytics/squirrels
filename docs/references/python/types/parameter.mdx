---
title: "Parameter"
tag: "class"
description: "Abstract base class for parameters"
---

Abstract base class for all parameter widget classes.

This is an abstract class and should not be instantiated directly. Use one of the concrete implementations:
- `SingleSelectParameter`
- `MultiSelectParameter`
- `DateParameter`
- `DateRangeParameter`
- `NumberParameter`
- `NumberRangeParameter`
- `TextParameter`

See the `parameters` module for concrete implementations.

Enumeration of data source types for parameter options.

## Values

### CONNECTION

Data from an external database connection.

**Value:** `"connection"`

### SEEDS

Data from CSV seed files.

**Value:** `"seeds"`

### VDL

Data from the Virtual Data Lake (built models).

**Value:** `"vdl"`

---
title: "NumberRangeDataSource"
tag: "class"
description: "Lookup table for number range parameter options"
---

Data source class for populating number range parameter options from a database table or query.

## Constructor

### Arguments

<ResponseField name="table_or_query" type="str" required>
  Either the name of the table to use, or a query to run
</ResponseField>

<ResponseField name="default_lower_value_col" type="str" required>
  The column name for the default lower value
</ResponseField>

<ResponseField name="default_upper_value_col" type="str" required>
  The column name for the default upper value
</ResponseField>

<ResponseField name="min_value_col" type="str | None" default="None">
  The column name for the minimum value
</ResponseField>

<ResponseField name="max_value_col" type="str | None" default="None">
  The column name for the maximum value
</ResponseField>

<ResponseField name="increment_col" type="str | None" default="None">
  The column name for the increment/step value
</ResponseField>

<ResponseField name="source" type="SourceEnum" default="SourceEnum.CONNECTION">
  The source to fetch data from
</ResponseField>

<ResponseField name="user_group_col" type="str | None" default="None">
  The column name of the user group for option visibility
</ResponseField>

<ResponseField name="parent_id_col" type="str | None" default="None">
  The column name of the parent option id for cascading
</ResponseField>

<ResponseField name="connection" type="str | None" default="None">
  Name of the connection to use
</ResponseField>
